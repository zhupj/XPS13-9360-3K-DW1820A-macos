## 二、这个EFI的使用方式

1. 用小兵的教程和镜像安装mac os
> g.daliansky.net/macOS-Catalina-10.15.1-19B88-Release-version-with-Clover-5098-original-image-Double-EFI-Version.html

2. 下载这个efi覆盖原来的efi即可。
   这样1820A 网卡也不用折腾可。
   
3.   Clover Configurator  工具。 改各种参数用的。
